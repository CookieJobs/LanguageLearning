## 路由目标

* 登录后统一进入首页仪表盘，不再直接进入学习页。

* 首次或更换学段时进入 Onboarding 过渡页，完成选择后回到首页。

* 首页统一展示入口：开始学习、连续打卡、掌握单词；更换学段跳转 Onboarding。

* 学习页按 URL 学段参数加载会话；复习页可切换“全部/当前学段”。

## 路由表（规范化）

* 公共：

  * `/login` → 登录页（未登录用户可访问）

* 受保护（需登录）：

  * `/onboarding` → 新用户/更换学段过渡页（minimal，无打卡/掌握）

  * `/` → 首页仪表盘（AuthGuard + LevelGuard）

  * `/learn/:level` → 学习页（AuthGuard；自校验参数并按需 `handleLevelSelect`）

  * `/review` → 复习页（AuthGuard；支持“全部/当前学段”切换）

  * `*` → 重定向至 `/`

## 守卫与加载逻辑

* `AuthGuard`（frontend/router/AuthGuard.tsx）

  * 无 `token` → 重定向 `/login`

* `LevelGuard`（frontend/router/LevelGuard.tsx）

  * 等待 `meLoaded` 后判断；`level` 为空 → 重定向 `/onboarding`

  * 仅包裹首页 `/`；`/learn/:level` 由页面自校验参数触发加载

* `AppContext`（frontend/contexts/AppContext.tsx）

  * 登录后加载 `/api/me` 与 `/api/stats/me`；

  * 有 `educationLevel` → 设置 `level` 并写入 `linguaCraft_lastLevel`

  * 无 `educationLevel` → 用本地 `linguaCraft_lastLevel` 兜底并 `PATCH /api/me` 回填；设置 `meLoaded`

## 导航规则

* 登录成功（frontend/pages/LoginPage.tsx）

  * 有或缺失学段都统一 `navigate('/')`；缺失学段由 `LevelGuard` 跳转至 `/onboarding`

* 更换学段（frontend/pages/HomePage.tsx）

  * 点击“更换学段” → `navigate('/onboarding')`

* Onboarding 完成选择（frontend/pages/OnboardingPage.tsx）

  * `updateMe` + 本地 `levelSince/lastLevel` → `navigate('/')`

* 头部 Logo（frontend/App.tsx）

  * 仅 `navigate('/')`；不清空 `level`

## 页面职责

* 首页（frontend/pages/HomePage.tsx）

  * 始终仪表盘视图：展示当前学段、打卡天数、累计掌握数量；“开始学习”“查看已掌握”“更换学段”

* 学习页（frontend/pages/LearnPage.tsx）

  * 读取 `:level` 参数；当队列结束显示 `SessionSummary`（掌握数、打卡增量、返回首页/开始下一次学习）

* 复习页（frontend/pages/ReviewPage.tsx + components/ReviewList.tsx）

  * 默认显示全部掌握；支持切换到“当前学段”（按 `levelSince` 过滤）

* Onboarding（frontend/pages/OnboardingPage.tsx + components/Onboarding.tsx）

  * minimal 模式，隐藏打卡/掌握徽章；仅用于新用户/更换学段过渡

## 具体改动点（一致化与去重）

1. `frontend/App.tsx` 路由表统一为上述结构；确保 `/` 用 `AuthGuard + LevelGuard` 包裹。
2. `frontend/pages/LoginPage.tsx` 登录后统一进入 `/`，去除直接跳 `learn/:level` 的逻辑。
3. `frontend/router/LevelGuard.tsx` 保持等待 `meLoaded` 再判定；仅用于 `/`。
4. `frontend/pages/HomePage.tsx` 移除内联“更换学段弹窗”，统一导航到 `/onboarding`。
5. `frontend/pages/LearnPage.tsx` 自校验参数，按需触发 `handleLevelSelect`，会话结束显示总结。
6. `frontend/components/ReviewList.tsx` 保留“全部/当前学段”切换按钮。

## 验证流程

* 新用户：`/login` → `/onboarding` → 选择学段 → `/` 仪表盘

* 老用户：`/login` → `/` 仪表盘 → “开始学习” → `/learn/:level`

* 更换学段：`/` → “更换学段” → `/onboarding` → 选择后 `/`

* 复习页：`/review` 显示全部/当前学段切换
